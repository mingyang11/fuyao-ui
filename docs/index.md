## 特性

- 🎨 基于 React 的组件库
- 📦 支持按需加载
- 🚀 支持 TypeScript
- 💪 支持主题定制

# 迷你组件库

```jsx
import { FuyaoButton } from 'fuyao-ui';

export default () => (
  <div>
    <FuyaoButton type="primary">Primary Button</FuyaoButton>
  </div>
);
```
